WriXt

WriXt is a fork of Writ CSS.

Simplistic, responsive, classless styles for semantic HTML with inbuilt dark mode support.

  • GitHub
  • CSS
  • Minified
  • Reference Page

Usage

Just copy the line below to the head tag of you HTML file

<link rel="stylesheet" href="https://wrixt.hrst.in/wrixt.min.css">

or download the full template here.

Goals

These were the original goals of writ:

  • Drop-in stylesheet
    • No classes
    • No extra elements
  • Responsive, without special-casing
  • Visually pleasing

They remain the same with wrixt.

Motivation

WriXt was born by extending Writ to support dark mode. I implemented dark mode using the latest CSS function light-dark.

The original motivation of Writ was this:

HTML is super easy to write. Markdown is even easier to write and generate HTML from. But what does it look like?

Not very nice, and it gets worse the more there is. Applying the Writ stylesheet makes it better.

Caveats

Writ was designed for modern, standards compliant browsers only. Same goes for WriXt. I have used the very new light-dark CSS function. So again, there are no compatibility hacks. Some things might (should not, i guess) break.
